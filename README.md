# N++

NHKラジオのWeb版にキーボードショートカットを追加します。

## 操作方法

再生エリアをクリックし、フォーカスが当たっている必要があります。

| Key | Control |
| ---- | ---- |
| Space | 再生 / 一時停止 |
| 左矢印 / 右矢印 | 5秒戻す / 5秒送る |
| 上矢印 / 下矢印 | 音量を10%上げる / 10%下げる |
